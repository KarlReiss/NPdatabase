# 下一步开发计划（后端完成后的阶段）

**当前状态**: 后端核心 API 已完成并通过验证（详见 `docs/backend-dev-log.md`）

---

## 1. 前端联调（优先级最高）

**目标**: 前端原型对接真实 API，完成核心流程可用。

- [ ] 明确前端框架与目录结构（沿用原型或重建）
- [ ] 对接列表与详情页 API：`/api/natural-products` / `/api/targets`
- [ ] 对接搜索与统计：`/api/search` / `/api/stats`
- [ ] 完成筛选 UI 与参数映射（mw/xlogp/psa/活性/毒性）
- [ ] 完成分页与排序规则（后端已实现稳定排序）
- [ ] 浏览器侧性能与分页体验优化

**验收方式**: 前端页面可通过真实数据完成“搜索 → 列表 → 详情 → 关联跳转”。

## 2. 数据质量与字段补齐

**目标**: 处理缺失字段，提高展示质量。
